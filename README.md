Dice Roller App

About
This is a beginner Android project built using Kotlin and Jetpack Compose.
The app simulates rolling a dice and shows a random number from 1 to 6 when the user taps a button.

Why I Built This
I created this project to practice Android fundamentals such as:
• State management
• Button click handling
• Random number generation
• Updating UI using Jetpack Compose

Features
• Roll dice with one button click
• Displays numbers from 1 to 6
• Clean and simple user interface
• Built fully using Compose

Tech Stack
• Kotlin
• Jetpack Compose
• Android Studio

How to Run

Clone the repository

Open in Android Studio

Run on emulator or Android device
